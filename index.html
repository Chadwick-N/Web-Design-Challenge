Summary: Latitude vs. X

An analysis was done on over 500 cities using OpenWeatherMap API to assemble the dataset.

Matplotlib was used to build visualizations on weather vs. latitude focusing on temperature, cloudiness, wind speed, and humidity.

This webpage will guide you through the visualizations that were made along with the analysis.
